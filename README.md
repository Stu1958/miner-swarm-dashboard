## ✨ Features
- **Fleet View:** Monitor up to 8 miners simultaneously on one screen.
- **Hardware Optimized:** Custom-built for the **Cheap Yellow Display (ESP32-CYD)**.
- **Avalon Persistence:** Automatically saves "Best Difficulty" stats to non-volatile storage (NVS) for Avalon Nano 3/3S.
- **Smart Detection:** Automatically adjusts logic based on your miner's hostname.

## 📏 Naming Requirements
To ensure the dashboard displays correctly, keep your miner names to **8 characters or less**. Names are case-insensitive.

| Miner Model | Keyword Rule | Example Name |
| :--- | :--- | :--- |
| **Avalon Nano 3S** | Must contain `NANO` and `3S` | `N3S_RIG` |
| **Avalon Nano 3** | Must contain `NANO` and `3` | `N3_MAIN` |
| **N-Series (NX/NQ)** | Must START with `NX` or `NQ` | `NX6_01` |
| **Standard Bitaxe** | Any other standard name | `Baxe_01` |
