📏 App Setup Naming (8 Chars Max)

When adding miners in the app setup, use these names to apply the correct logic. Names are case-insensitive.

Avalon 3S: Use NANO & 3S in name
N-Series: MUST start with NX or NQ
Bitaxe/Other: Use any other name (e.g. RIG_01)
